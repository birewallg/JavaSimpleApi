# Http Api

- Create database
    HEAD/app/api/new
- Create user
    GET/api/new/{login}?name=string&date=long&age=int
- Show users 
    GET/app/api/users 
    GET/app/api/users/{login}
- Edit user metadata 
    GET/api/edit/{login}?name=string&date=long&age=int 
- Delete user from database 
    DELETE/api/edit/{login}
